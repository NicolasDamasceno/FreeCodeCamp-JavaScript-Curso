# FreeCodeCamp-JavaScript-Curso
Repositório dedicado a centralizar os exercícios, anotações e projetos desenvolvidos durante a certificação de **JavaScript Algorithms and Data Structures** do FreeCodeCamp.

---

## 🚀 Progresso do Curso
- [ ] Introdução ao JavaScript Básico
- [ ] Programação Orientada a Objetos (POO)
- [ ] Programação Funcional
- [ ] Algoritmos e Estruturas de Dados
- [ ] Projetos Finais (Certificação)

## 📂 Organização do Repositório

O repositório está dividido por módulos para facilitar a navegação:

* `/basico`: Fundamentos como variáveis, loops e condicionais.
* `/algoritmos`: Soluções para os desafios de lógica de programação.
* `/projetos-finais`: Os 5 projetos necessários para obter o certificado.

## 🛠️ Tecnologias e Ferramentas
* **Linguagem:** JavaScript (ES6+)
* **Ambiente de Execução:** Node.js / Browser Console
* **Editor:** VS Code
* **Sistema Operacional:** Linux Mint

## 📝 Como executar os códigos
Para rodar os scripts localmente, certifique-se de ter o Node.js instalado e execute:
```bash
node nome-do-arquivo.js
